# CodeAlpha_Data-Visualization
# Data Story
# Problem

Banks need to identify high-risk customers, reduce churn, detect fraud, and improve profitability.

# Findings
Certain risk ratings contribute disproportionately to NPAs.
Dormant customers show higher churn probability.
Some transaction channels exhibit elevated fraud rates.
Significant missing values exist in lending and digital banking fields.
Outliers in income, balance, and loan amounts require treatment before modeling.

# Recommendations
Target high-risk customers for proactive monitoring.
Launch retention campaigns for dormant accounts.
Strengthen fraud controls on high-risk channels.
Perform missing-value imputation and outlier treatment before ML modeling.
Build predictive models for Churn, Fraud Detection, and Loan Default.
